About GamepadTester.tools

A lightweight browser tool for checking controller input.

GamepadTester.tools is a static website that helps users inspect button presses, trigger pressure, analog stick movement, deadzone behavior, and drift signals using the browser Gamepad API.

How it works

The site reads controller state directly from supported browsers. There is no account system, downloadable client, or server-side controller processing.

Who it helps

It is useful for players troubleshooting drift, checking trigger range, validating mappings, or confirming a controller works before deeper hardware repair.

What the site focuses on

GamepadTester.tools is designed to stay simple, fast, and understandable for everyday controller checks.

Browser-first testing

The site relies on standard browser APIs so users can test controllers without installing native software.

Live visual feedback

Buttons, sticks, triggers, and drift indicators update in real time to make hardware issues easier to spot.

Focused utility pages

Dedicated pages for drift, deadzone, Xbox, and PS5 checks make it easier to run a specific test without extra clutter.

No promises of hardware repair

The site is a diagnostic aid only. It does not repair devices or guarantee compatibility with every browser-controller combination.
