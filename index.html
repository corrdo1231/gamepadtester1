What Can You Test?

🔘 All Buttons

Every face button, shoulder, trigger, D-pad direction, stick click, and system button lights up when pressed.

🕹️ Analog Sticks

Real-time XY axis position shown as a moving dot with numeric readout. Spot deadzone gaps and erratic response.

📉 Stick Drift

Release all sticks and watch for non-zero readings. Any value above 2% at rest indicates potential drift.

📳 Vibration

Test left and right rumble motors at different intensities. Requires Chrome or Edge on Windows.

⚡ Input Latency

Measures milliseconds between a button press and browser detection. Compare wired vs Bluetooth lag.

🎮 Multi-Controller

The browser Gamepad API supports up to 4 simultaneous controllers. Refresh to switch active pad.

Supported Controllers

PlayStation

PS5 DualSense, PS4 DualShock 4, PS3 controllers via USB or Bluetooth.

Xbox

Xbox Series X/S, Xbox One, Xbox 360 controllers (wired or via adapter).

Nintendo

Switch Pro Controller, Joy-Con (paired) via USB or Bluetooth.

Generic

Any USB or Bluetooth HID gamepad that exposes standard Gamepad API axes and buttons.
