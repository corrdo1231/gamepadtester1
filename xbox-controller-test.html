Xbox controller checklist

A quick pass through these inputs usually tells you whether the controller is ready for games or streaming software.

Face buttons

Tap A/B/X/Y rapidly and make sure the active state changes immediately without missed presses.

Trigger sweep

Pull LT and RT slowly from rest to full press. The values should climb smoothly toward 1.000.

Stick return

Roll both sticks to the edge, release them, and watch whether they return to center without persistent offset.
